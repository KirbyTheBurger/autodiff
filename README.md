A program that gets the derivative of any function at a specific point, utilizing dual numbers, a complex set of numbers of the form $a + b\varepsilon$ where $\varepsilon^2 = 0$ but $\varepsilon \neq 0$.  

## Installation  
To install autodiff, make sure you have cargo downloaded (Rust's package manager), and then simply run the command
```bash
cargo install autodifferentiate
```  
*The name autodiff was sadly already taken*
